# my-study-mysql-2025

## はじめに

SQL構文を勉強するために作成したリポジトリ。

## 環境

- Windows 11
- Visual Studio
- MySQL Server 9.1
- MySQL Workbench 8.0

## 起動コマンド

コマンドプロンプトの場合、管理者として実行してからやる

```bash
net start MySQL91
```

## MySQLにログイン

``` bash
mysql -u root -p
```
